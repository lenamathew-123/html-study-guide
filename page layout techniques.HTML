Page Layout Techniques

There are three tags.

  • < header >
  • < Main >
  • < footer >

Inside main tag

There are three sub tags

  1. Section Tag

    2. For a section on your page

    < section >

  2. Article tag

    3. For an article on your page

    <article>

  3. Aside tags

    4. For content aside main contents

    <aside>
